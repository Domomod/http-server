Detailed Description

Http request representation.

This class allows for represenation of a http/1.1 request.
Because the header keys may vary from request to request, this class stores header keys and
vectors of coresponding values in a dictionary. Both values and keys are stored as strings.
Conversion of those values is left to the class user.
